# 欢迎使用 Obsidian 自知知识库

这是一个由 Codex 配置的 Obsidian 知识库。

## 功能特性

- **Git 同步**：自动备份到 GitHub
- **插件管理**：日历、任务、图表等功能
- **模板系统**：快速创建笔记

## 快速开始

1. 打开 Obsidian，打开这个知识库
2. 在设置中启用社区插件
3. 搜索并安装以下插件：
   - Obsidian Git
   - Calendar
   - Tasks
   - Dataview

## 配置步骤

### 1. 配置 Git 同步

在 Obsidian 设置中：
1. 启用 Obsidian Git 插件
2. 设置自动备份间隔
3. 配置 GitHub 仓库

### 2. 配置 GitHub（可选）

如果需要云端备份：
1. 在 GitHub 创建新仓库
2. 在 Git 插件中设置仓库地址
3. 配置 Git 凭据
