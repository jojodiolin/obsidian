# 20-知识库 · LLM 编译的知识库

**这是 LLM 的领地**。人很少直接编辑这里的内容 — 由 `/编译` 命令增量维护。

## 目录结构

| 子目录 | 内容 | 颗粒度 |
|--------|------|--------|
| `INDEX.md` | 总索引 + 全部概念摘要 | 总览 |
| `概念/` | 原子概念笔记（一个概念一篇） | 最细 |
| `主题/` | 主题长文（多概念聚合） | 中 |
| `人物/` | 人物卡 | 中 |
| `摘要/` | 原文摘要 + 反向链接到 raw | 中 |

## 编译规则

1. **新增**：原始资料中出现新概念 → 在 `概念/` 新建笔记
2. **更新**：已有概念有新进展 → 在笔记末尾追加"## 更新 YYYY-MM-DD"段落
3. **链接**：所有笔记之间用 `[[概念/xxx]]` 双链，不用裸路径
4. **索引**：每次编译后刷新 `INDEX.md`
