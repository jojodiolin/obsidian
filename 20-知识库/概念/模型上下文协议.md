# 模型上下文协议 (MCP)

## 一句话定义
Anthropic 2024 年 11 月推出的开放标准，规范 LLM 连接外部工具和数据源的方式。

## 详细说明
MCP 的核心价值不只是"工具调用"——Sean Lynch 指出其**真正的架构优势**在于：将**认证流程隔离在智能体的上下文窗口之外**。

传统技能或 CLI 集成通常要求智能体在上下文中处理认证步骤，这既浪费宝贵的 token 空间，又存在安全风险。MCP 让认证逻辑留在 MCP 服务器侧，对智能体只暴露纯工具接口。

理想化形式：MCP 服务器甚至可以**仅仅是 API 的认证网关**——只做身份代理，不传认证细节给智能体，仍然是一个胜利。

## 关键事实
- 推出方：Anthropic
- 推出时间：2024 年 11 月
- 核心收益：减少上下文窗口污染 + 提升安全性
- 适用场景：生产环境部署的 AI 智能体
