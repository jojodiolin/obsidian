# MosaicLeaks

## 一句话定义
通过组合多次看似无害的查询，利用马赛克效应从 LLM 研究代理中提取机密的新型攻击。

## 详细说明
ServiceNow 研究团队提出的攻击方法，针对**使用搜索引擎/数据库等工具的 LLM 研究代理**。

### 马赛克效应
单个查询看似无害，但**多个查询汇总后**能重建机密信息——就像马赛克拼图，单块看不出什么，拼起来却暴露全貌。

### 攻击流程
1. 攻击者向 LLM 代理发送多个**单看无害**的多跳问题
2. 代理使用工具逐个查询
3. 攻击者从聚合回复中**反推机密**

### 评估基准
研究团队创建了 **1001 个多跳问题**的基准测试集来评估代理的脆弱性。

## 关键事实
- 提出方：ServiceNow + Hugging Face
- 攻击目标：LLM 研究代理
- 利用原理：马赛克效应
- 评估规模：1001 多跳问题
- 相关论文：arXiv 2605.30727
